# Spring Boot 第一节课项目总结

## 项目概览

本项目是一个最简单的 Spring Boot Web 应用，启动后访问 `/hello` 路径即可返回 `"hello spring boot"` 字符串。

- **项目名称**: springboot
- **技术栈**: Spring Boot 3.0.5 + Java 17 + Maven
- **服务端口**: 8081

---

## 实现步骤

### 步骤一：创建 Maven 项目并配置 pom.xml

在项目根目录 `springboot/` 下创建 `pom.xml`，引入 Spring Boot 父项目和 `spring-boot-starter-web` 依赖。

**文件路径**: `springboot/pom.xml`

```xml
<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>

    <groupId>org.wanghong</groupId>
    <artifactId>springboot</artifactId>
    <version>1.0-SNAPSHOT</version>

    <!-- 继承 Spring Boot 父项目，统一管理版本 -->
    <parent>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-parent</artifactId>
        <version>3.0.5</version>
    </parent>

    <dependencies>
        <!-- Web 启动器：内嵌 Tomcat + Spring MVC -->
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-web</artifactId>
        </dependency>
    </dependencies>

    <properties>
        <maven.compiler.source>17</maven.compiler.source>
        <maven.compiler.target>17</maven.compiler.target>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    </properties>
</project>
```

**关键点**：
- `<parent>` 标签让本项目继承 `spring-boot-starter-parent`，省去大量版本声明。
- `spring-boot-starter-web` 是 Web 开发的核心起步依赖，内含 Spring MVC 和内嵌 Tomcat。

---

### 步骤二：编写 Spring Boot 启动类

在 `src/main/java/org/wanghong/` 目录下创建 `MainApplication.java`。

**文件路径**: `springboot/src/main/java/org/wanghong/MainApplication.java`

```java
package org.wanghong;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication  // 复合注解：@SpringBootConfiguration + @EnableAutoConfiguration + @ComponentScan
public class MainApplication {
    public static void main(String[] args) {
        SpringApplication.run(MainApplication.class, args);
    }
}
```

**关键点**：
- `@SpringBootApplication` 是一个**复合注解**，等价于同时标注了 `@SpringBootConfiguration`、`@EnableAutoConfiguration` 和 `@ComponentScan`，一个注解即可完成自动配置和组件扫描。
- `SpringApplication.run()` 是 Spring Boot 应用的入口，它会启动内嵌 Tomcat 并初始化 Spring 容器。

---

### 步骤三：编写 Hello 控制器

在 `src/main/java/org/wanghong/controller/` 目录下创建 `HelloController.java`。

**文件路径**: `springboot/src/main/java/org/wanghong/controller/HelloController.java`

```java
package org.wanghong.controller;

import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController  // 复合注解：@Controller + @ResponseBody
public class HelloController {

    @GetMapping("hello")
    public String hello() {
        return "hello spring boot";
    }
}
```

**关键点**：
- `@RestController` 是一个**复合注解**，等价于 `@Controller` + `@ResponseBody`，表明该类中所有方法的返回值都会直接写入 HTTP 响应体（而非跳转页面）。
- `@GetMapping("hello")` 是 `@RequestMapping(method = RequestMethod.GET)` 的简写形式，将 `/hello` 路径的 GET 请求映射到 `hello()` 方法。

---

### 步骤四：配置服务端口

在 `src/main/resources/` 目录下创建 `application.properties`，将默认端口 8080 改为 8081。

**文件路径**: `springboot/src/main/resources/application.properties`

```properties
server.port=8081
```

**关键点**：Spring Boot 的自动配置会读取 `application.properties`，`server.port` 是内嵌 Tomcat 的端口配置项。

## 运行与验证

1. 在 IDEA 中运行 `MainApplication` 的 `main` 方法，或执行 Maven 命令：
   ```bash
   mvn spring-boot:run
   ```
2. 打开浏览器访问 `http://localhost:8081/hello`，页面显示：
   ```
   hello spring boot
   ```

## 核心概念总结

| 概念 | 说明 |
|------|------|
| `@SpringBootApplication` | 复合注解，一行替代三个注解，开启自动配置和组件扫描 |
| `@RestController` | 复合注解，所有方法返回值自动序列化为 JSON/文本写入响应体 |
| `@GetMapping` | 将 HTTP GET 请求映射到指定方法 |
| `spring-boot-starter-web` | Web 场景启动器，集合了 Spring MVC + Tomcat 等依赖 |
| `application.properties` | Spring Boot 应用的默认配置文件 |
